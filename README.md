
# benford-physics-corpus

Python pipeline for testing Benford’s Law in physics papers using numerical extraction, first-digit analysis, and statistical comparison.

## Project summary

This repository explores whether numerical values in physics papers follow Benford’s Law, the logarithmic distribution often found in naturally occurring numerical datasets.
